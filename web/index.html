Cách dùng

Bắt đầu trong
3 bước đơn giản

Cài đặt → Nạp tài liệu → Hỏi đáp ngay lập tức.

Bước 01
Cài đặt & Cấu hình

Clone repo, chạy uv sync và tạo file .env với API key của bạn.

Bước 02
Nạp tài liệu

Trỏ đến thư mục chứa .txt/.md. Hệ thống tự động băm và đưa vào ChromaDB.

Bước 03
Hỏi đáp thông minh

Dùng docchat ask hoặc docchat chat để trò chuyện với tài liệu của bạn.

Bonus
Đánh giá chất lượng

Chạy RAGAS evaluation để kiểm tra độ tin cậy và xuất báo cáo CSV chi tiết.

CLI Reference

Bốn lệnh
cốt lõi

Mọi tính năng đều truy cập qua giao diện dòng lệnh trực quan.

docchat index
Nạp tài liệu vào DB

Đọc và băm file .txt/.md vào ChromaDB. Hỗ trợ cờ --embedder openai|local.

docchat ask
Hỏi nhanh một câu

Single-turn query với streaming output. Kết quả xuất ra từng chữ theo thời gian thực.

docchat chat
Chế độ hội thoại liên tục

Multi-turn REPL với bộ nhớ ngữ cảnh. Hỗ trợ /clear, /stats, /exit bên trong.

docchat info
Kiểm tra trạng thái DB

Xem số lượng chunks đã index, cấu hình embedder và sức khoẻ ChromaDB.

Kiến trúc

Mô-đun
rõ ràng, dễ mở rộng

Mỗi file có trách nhiệm riêng biệt — dễ đọc, dễ tuỳ chỉnh.

loader.py

Đọc và quét sâu cây thư mục, phát hiện file .txt và .md.

chunker.py

Băm dữ liệu theo Token Limit, bảo vệ dung lượng context window.

store.py

ChromaVectorStore, BM25, RRF fusion, Cross-Encoder Reranker node.

embedder.py

Factory sinh mã Embedding — hỗ trợ OpenAI và SentenceTransformers local.

llm.py

Đếm token, quản lý Memory, kết nối OpenAI / Anthropic API.

prompt_manager.py

Prompt Tiếng Việt lưu qua YAML — dễ chỉnh sửa không cần sửa code.
